# iARAP

## Get the code

Clone the project

```
git clone --recursive https://gitlab.cg.tu-berlin.de/yugon/intrinsic-arap.git
```

## Build the code

Configure with cmake and compile e.g.

```
cd intrinsic-arap
mkdir build
cd build
cmake ..
make -j6
```

# Run the code
Start by e.g. 
```
./iARAP_gui ../bunny.off
```
